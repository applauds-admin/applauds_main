# Applauds Contracts

## Environment Variable

### View / Set

```shell
# set password
npx env-enc set-pw

# view all envs
npx env-enc view

# set envs
npx env-enc set
```

### Required Environment Variable

- _PRIVATE_KEY_ for your development wallet
- _POLYGON_MUMBAI_RPC_URL_ for the network that you intend to use
- _POLYGONSCAN_API_KEY_ for verify contracts
